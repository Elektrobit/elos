[source]

Function: elosEventFilterClone¶

Description: Duplicates elosRpnFilter into a newly allocated

memory region. The copy is completely independent of the original and needs to be freed separately.

Input: Two elosRpnFilter pointers (from and to) Info: ‘to’ needs to be freed with elosEventFilterDelete()

Return: - FILTER_RESULT_OK on success,

  • FILTER_RESULT_ERROR on failure.
